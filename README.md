# Writeups

This repository is my growing collection of security challenge writeups across multiple platforms.  
I use it as both a learning log and a public portfolio: each writeup captures my approach, what worked, what failed, and the security lessons that stood out.

The goal is to keep writeups practical, readable, and easy to navigate for recruiters, peers, and anyone learning through hands-on labs.
